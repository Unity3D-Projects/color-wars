LOGLINE: Retro-futuristic sportsball

CONTROLS & GAMEPLAY:
- Move around/aim ...........................................left stick
- Shoot ball ..............................................left trigger
- Parry (deflect) ball (if ball is very close)........................A
- Dash (Hold to charge)...................................right trigger
- Pause ..........................................................start
- Reset game (only available in pause screen).................dpad down

GOAL:
- Goal periodically changes color
- Points awarded to team with current goal color
- Most points wins
